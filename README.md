## Getting Started

먼저, 아래 명령어로 해당 repo를 clone해주세요
<br>     

```bash
 git clone https://github.com/donghyunami/papel-next
```

<br><br>

그 다음, 설치해주세요
<br>  
```bash
 npm install 
 #또는 
 yarn install
```

<br><br>  
설치했다면 서버를 실행해주세요
<br>  
```bash
npm run dev
 #또는
yarn dev
```  

당신의 브라우저를 열어서  http://localhost:3050 으로 접속해주세요
